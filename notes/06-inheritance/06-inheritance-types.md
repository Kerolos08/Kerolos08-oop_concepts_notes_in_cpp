# Common Used / Supported Inheritance types

## 1- Single Inheritance

One **base class** → one **derived class**.

## 2- Multi Level Inheritance

A class inherits from another **derived class** — a chain of inheritance.

## 3- Hierarchical Inheritance

One **base class** is inherited by **multiple derived classes**.

# Barley Used / Not Supported Inheritance types

## 1- Multiple Inheritance

One **derived class** inherits from **two or more base classes**.

## 2- Hybrid Inheritance

A **combination** of two or more of the above types.

### Those inheritances are not preferred for many reasons like:

- **Not supported by modern programming languages.**
- **Barley used.**
- **Replicated Members / Methods.**
- **Constructors Issues.**
- **Compiler Issues.**
